#I started software engineering because technology has always fascinated me and how it impacts the world. Also, I like how complicated coding sounds and I like to challenge myself to learn them. There are lots of coding languages to learn from and I would like to learn some of them. It brings me great excitement to possibly work for big companies like Apple or Google. The fact that you don't need a college degree for this also brought to become interested in software engineering. I would like to pursue a career in creating video games using coding. I hope to learn a lot by joining Flatiron School and to pursue my goal to become a software engineer. There’s a lot of languages that I would like to learn to program because I would also want to design video games even though it usually uses C++. Currently, I’m a college student in New York City and would like to learn about coding in more detail so I joined Flatiron to learn more about coding. Since college gives only a general idea in computer science, a coding boot camp goes more into depth in specific languages used in everyday programming. If I learn the basics of Ruby and Java, then I can learn other languages since they are all similar. In college, it would be kind of confusing so I felt that in a boot camp, I would attain all the knowledge needed to code and become a software engineer. Besides, out of all other possible careers, becoming a software engineer has one of the best salaries and you don’t need a college degree (even though I still plan on getting mine). These are only a few of the reasons of why I want to pursue software engineering.
